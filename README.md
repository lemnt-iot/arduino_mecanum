# 麦克纳姆轮智能小车项目

## 项目简介
该项目使用Arduino主板 + PS2手柄 + L298N 来控制设备的移动。通过读取 PS2 控制器的模拟信号（X轴和Y轴），实现设备的上下左右控制。

## 功能
- 根据右摇杆的不同方向控制设备：
  - **上**：设备向前移动
  - **下**：设备向后移动
  - **左**：设备向左移动
  - **右**：设备向右移动
  - **中立**：设备停止

## 代码结构
- `PS2xControl::controller()`：负责处理控制逻辑，根据摇杆的X轴和Y轴数值，输出相应的控制指令。
   
示例：
```cpp
void loop() {
    PS2xControl::controller();
}
